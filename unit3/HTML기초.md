## 체크포인트
- HTML -> 구조,  CSS -> 스타일, JavaScript -> 상호작용
- 중요하고, 헷갈렸던 태그 
html    - html 문서, 언어지정
a       - anchor 하이퍼링크 지정
p       - pragraph 단락영역
header  - 헤더영역 (제목, 검색창 등 상단에 기록되는 내용 지정)
h1 ~ h6 - h1부터 글자 크기가 점점 줄어들며 가장 큰 h1은 제목으로 사용
link    - 외부 css와 연결
- 로그인 페이지 만들기 실습 (login.html & style.css 사용)
- 시멘틱 요소를 활용해 검색적인 부분과 코딩 내용적 정리 부분 효율성 증가
- !! 태크를 열었으면 다시 태그 닫아주기 !!
